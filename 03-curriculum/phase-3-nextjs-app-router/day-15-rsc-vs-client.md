### Server Component

Digunakan untuk:

* Product list awal.
* Detail produk.
* Transaction list.
* Dashboard summary.

### Client Component

Digunakan untuk:

* Cart.
* Search interaktif.
* Filter interaktif.
* Modal.
* Form dengan interaksi client-side.

Jangan menambahkan `"use client"` pada seluruh page.
